## 潮玩启东模块开发计划

基于"美食启东"模块的实现模式，开发"潮玩启东"管理模块。

### 一、模块结构

```
cms/
├── models/
│   └── TravelCategory.php      # 栏目模型
│   └── TravelItem.php          # 内容模型
├── controllers/
│   └── TravelCategoryController.php  # 栏目管理控制器
│   └── TravelItemController.php      # 内容管理控制器
│   └── TravelApiController.php       # H5 API接口控制器
├── views/
│   └── travel/
│       ├── categories/
│       │   ├── index.php      # 栏目列表页
│       │   └── form.php       # 栏目表单页
│       └── items/
│           ├── index.php      # 内容列表页
│           └── form.php       # 内容表单页（富文本编辑器）
└── sql/
    └── travel_schema.sql       # 数据库初始化脚本
```

### 二、数据库设计

**travel_categories 表**
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INT AUTO_INCREMENT | 主键 |
| code | VARCHAR(50) UNIQUE | 栏目编码 |
| name | VARCHAR(100) | 栏目名称 |
| sort_order | INT DEFAULT 0 | 排序 |
| created_at | DATETIME | 创建时间 |
| updated_at | DATETIME | 更新时间 |

**预置栏目：** 文旅景点 (culture_scenic)、乡镇景点 (town_scenic)

**travel_items 表**
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INT AUTO_INCREMENT | 主键 |
| category_id | INT NOT NULL | 关联栏目ID |
| title | VARCHAR(200) NOT NULL | 标题 |
| subtitle | VARCHAR(200) | 副标题 |
| cover_url | VARCHAR(500) | 封面图URL |
| content | MEDIUMTEXT | 正文（富文本HTML） |
| address | VARCHAR(255) | 地址 |
| phone | VARCHAR(50) | 联系电话 |
| business_hours | VARCHAR(100) | 营业时间 |
| ticket_price | DECIMAL(10,2) | 门票价格 |
| latitude | DECIMAL(10,6) | 纬度 |
| longitude | DECIMAL(11,6) | 经度 |
| is_recommended | TINYINT(1) DEFAULT 0 | 推荐 |
| is_pinned | TINYINT(1) DEFAULT 0 | 置顶 |
| sort_order | INT DEFAULT 0 | 排序 |
| is_active | TINYINT(1) DEFAULT 1 | 上线状态 |
| created_at | DATETIME | 创建时间 |
| updated_at | DATETIME | 更新时间 |

### 三、API 接口

| 接口 | 路径 | 说明 |
|------|------|------|
| 封面页 | GET /api/travel/cover | 每栏目返回6条推荐 |
| 列表页 | GET /api/travel/list?category_code=xxx | 分类内容列表 |
| 详情页 | GET /api/travel/detail?id=xxx | 内容详情 |

### 四、排序规则
**置顶 > 推荐 > sort_order > id DESC**

### 五、实施步骤

1. 创建数据库脚本 `travel_schema.sql`
2. 创建模型 `TravelCategory.php`, `TravelItem.php`
3. 创建控制器 `TravelCategoryController.php`, `TravelItemController.php`, `TravelApiController.php`
4. 创建视图（栏目/内容列表页、表单页）
5. 注册路由（后台管理 + API）
6. 更新侧边栏导航
7. 导入数据库
