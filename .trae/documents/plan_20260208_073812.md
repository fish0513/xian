# 首页模块开发方案

根据您的需求，我们将构建一个静态的、适配移动端的 H5 首页。

## 1. 目录结构规划
为了保持项目整洁，我们将新建 `home` 目录专门存放首页代码，不直接混在根目录。
建议路径：`d:\www\xian\home`

结构如下：
```
d:\www\xian\home\
├── index.html          # 首页入口文件
└── assets\
    ├── css\
    │   └── style.css   # 样式文件
    └── images\         # 图片资源
        ├── bg-home.jpg
        ├── title-home.png
        └── bg-button.png
```

## 2. 资源处理
将您提供的素材从 `home-Material` 迁移至 `home/assets/images`：
- `bg-home.jpg` -> 作为全屏背景
- `title-home.png` -> 页面顶部标题
- `bg-button.png` -> 导航按钮背景

## 3. 开发实现细节
- **技术栈**：原生 HTML5 + CSS3 (无需构建工具，轻量高效)。
- **移动端适配**：
  - 使用 `<meta name="viewport" ...>` 控制视口。
  - 使用 Flexbox 布局实现垂直居中和弹性分布。
  - 采用 `vh/vw` 或 `%` 单位确保在不同尺寸手机上显示效果一致。
- **页面交互**：
  - 静态页面，无动态数据加载。
  - 按钮链接配置（基于现有模块推断）：
    - **美食启东** -> 跳转至 `/food/`
    - **潮玩启东** -> 跳转至 `/travel/`
    - **宜居启东** -> 暂无链接 (或是敬请期待)
    - **乐购启东** -> 暂无链接 (或是敬请期待)

## 4. 部署与配置
修改 Nginx 配置文件 (`nginx.htaccess`)，将根路径 `/` 指向新的首页目录。

**新增配置示例**：
```nginx
location / {
    alias "D:/www/xian/home/";
    index index.html;
}
```
*注：这将接管网站根目录的访问。原有的 `index.php` 如果不再需要，可以保留作为备份。*
